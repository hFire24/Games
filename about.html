About this website

This website is developed by hFire24, who uses their programming skills to create simple games. So far, hFire24 has developed 3 games using Javascript.

  • Math Game: This is a simple math game inspired by Alarmy's math alarm. To play the game, the player has to solve addition and multiplication problems. The player could also type in the numbers while playing the game as input.
  • Rock Paper Scissors: This is a simple Rock Paper Scissors game with difficulty options and a score tracker.
  • Card Battle Stadium: This is a battle simulator with high levels of RNG. Players attack using cards. The player can choose to pull from different decks if they have enough MP. This was hFire24's first HTML game.
